## V2.4 — Sepet (uçtan uca)

### Mimari
- **Auth user:** `cart_items` tablosu (RLS ile own-cart erişimi).
- **Misafir:** `localStorage` (`auto-filter:cart`).
- **Login transition:** Misafir sepetinde ürün varsa, ilk authed mount'ta `mergeGuestCart` server action çağrılır → DB'ye merge edilir → localStorage temizlenir.

### Yeni dosyalar

**Backend / data:**
- `lib/cart/actions.ts` — server actions:
  - `addToCart(productId, qty)` — auth check, stok cap, `cart_items` upsert, `reserved_stock += qty` (admin client).
  - `updateCartQuantity(productId, qty)` — qty=0 ise sil, aksi halde update; reserved_stock delta uygulanır.
  - `removeFromCart` — `updateCartQuantity(_, 0)` proxy'si.
  - `getCart()` — embedded join ile `cart_items + products`.
  - `mergeGuestCart(items[])` — her item için `addToCart` çağrısı.
  - **Not:** `reserved_stock` güncellemesi sequential read-modify-write — race condition var. Production'da Postgres function ile atomic yap (TODO).
- `app/api/cart/products/route.ts` — POST `{ ids[] }` → ürün detayları (misafir sepeti hidrasyonu için).

**Auth helper:**
- `lib/auth/user.ts` — `getCurrentUser` (`React.cache` ile request başına 1 kez Supabase'e gider).

**Client cart provider:**
- `components/cart/CartProvider.tsx` — context + `useCart()` hook. Items, count, total, loading, isAuthenticated, addItem, updateQty, removeItem.
  - Auth: server action çağırır → sonra `getCart()` ile state'i tazeler.
  - Misafir: `localStorage` + `/api/cart/products` ile hidrasyon, lokal state mutasyonu.
  - Login transition'da otomatik merge.

**Header refactor:**
- `components/StoreHeader.tsx` (client) — paylaşılan header. Logo, Ürünler linki, **sepet ikonu + adet rozeti** (`useCart().count`), kullanıcı greeting + Hesabım + Çıkış / yoksa Giriş Yap + Kayıt Ol.
- `components/StoreHeaderShell.tsx` (server) — `getCurrentUser()` çekip client `<StoreHeader>`'a aktarır.
- Mevcut inline header'lar kaldırıldı: `app/page.tsx`, `app/(store)/urunler/page.tsx`, `app/(store)/urun/[...name]/page.tsx`.

**Root layout:**
- `app/layout.tsx` artık async — `getCurrentUser()` + (auth'sa) `getCart()` çekip `<CartProvider isAuthenticated={...} initialItems={...}>` ile sarmalıyor.

**Sepet sayfası:**
- `app/(store)/sepet/page.tsx` (server) — `<StoreHeaderShell />` + `<CartView />`.
- `app/(store)/sepet/CartView.tsx` (client) — `useCart()` ile çalışır:
  - Loading state, boş state ("Sepetiniz boş" + ürünleri keşfet linki).
  - Ürün satırları: resim, marka rozeti, kod + fancy name, adet ± (stok cap), kaldır (×), line total.
  - Sağ panel sipariş özeti: ara toplam, kargo placeholder, toplam, "Ödemeye Geç" — auth'sa `/odeme`, misafirse `/giris?next=/odeme` + bilgilendirme metni.

**AddToCart bağlantıları:**
- `app/(store)/urun/[...name]/AddToCart.tsx` — `useCart().addItem(productId, qty)` çağrısı + pending/done/error feedback.
- `app/(store)/urunler/AddToCartButton.tsx` (yeni) — listing kartları için tek tıkla 1 adet ekleme.
- `components/FilterWidget.tsx` — MannCard ve FiltronCard içinde `useCart()` kullanımı, `preventDefault/stopPropagation` ile kart linkinin tetiklenmesini engelleyerek.

### Proxy değişikliği
- `proxy.ts` matcher'dan `/sepet` çıkarıldı — misafirler de görebilsin. `/odeme` login zorunluluğu devam ediyor.

### Eski header stillerinin temizliği
- `urunler/page.tsx` ve `app/page.tsx`'teki kullanılmayan header/nav stilleri silindi.

## Bilinen TODO'lar (V2.4 sonrası)

1. **`reserved_stock` race condition** — sequential read-modify-write yerine Postgres function (SECURITY DEFINER) veya transaction + row lock. MVP için kabul edilebilir, prod öncesi yap.
2. **30-dakika expiry cron** — plan'da V2.4 listesinde ama yapılmadı. Vercel Cron + bir route handler ile: 30+ dk önce update edilmiş `cart_items` için `reserved_stock`'u geri ver.
3. **Catalog cache invalidation** — `/urunler` ve `/urun/[...name]` `unstable_cache(revalidate: 3600)` kullanıyor; admin'den fiyat/stok değişince otomatik yenilenmiyor. V2.6 admin paneli geldiğinde `revalidateTag("products")` action içine eklenmeli; ya da Supabase webhook → `/api/revalidate?tag=products` route'una.
